Sistemas de control de carga de combustibles refinados



Computadores de flujo marca Contrec para el control de carga de combustibles refinados. 

  • Entrada de pulsos para conectar a medidores de flujo.
  • Preselección del volumen de despacho (lote).
  • Registro de las 100 ultimas transacciones con referencia de fecha y hora.
  • Salida a impresora
  • Comunicación modbus para monitoreo desde cuarto de control.
  • Salida por relevadores para control de valvulas.
  • Entradas digitales para conectar sistema de sobrellenado y tierra.
